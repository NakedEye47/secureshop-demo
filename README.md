# SecureShop (Static Demo)

Welcome to the **SecureShop Demo**! 

> **⚠️ IMPORTANT NOTICE**
> 
> This repository contains a **purely static, frontend-only demonstration** of the SecureShop E-Commerce platform. It has been statically exported for portfolio viewing purposes and does not contain the backend source code, database architecture, or server-side authentication systems.
>
> All products, orders, and checkouts in this demo are simulated and run entirely on the client side using mock data.

## 🚀 Full-Stack Version
The complete, production-ready version of SecureShop is a robust, full-stack application built with:
- **Next.js 14** (App Router)
- **PostgreSQL & Drizzle ORM** (Database)
- **NextAuth.js v5** (Authentication)
- **PayMongo** (Payment Gateway Integration)
- **Tailwind CSS & shadcn/ui** (Styling)
